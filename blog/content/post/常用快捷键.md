iterm

```
command <- -> 左右切换
command t new tab
command w close
command q quit
```

chrome

1. command shift n -- incognito
2. command w close
3. command option <- -> 左右切换
4. command shift w close window

[man 命令的名称] 可以查看命令的详细用法

```
man htop
man ls
```

切换 app

- command + tab
- 四指拖动

快速显示启动台

- 四指聚拢
